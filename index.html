ISS Tracker

This script is an Automated ISS Tracking program. It concurrently monitors two external data streams: the real-time orbital position of the ISS and sunrise/sunset. When the ISS enters a defined coordinate radius of the user during hours of darkness, the script autonomously triggers an SMTP-based email notification to alert the user to look at the sky.

Cookie Clicker

A bot developed using Selenium to automate the "Cookie Clicker" game. The script simulates a user input by using WebDriver and CSS Selectors to perform real-time scraping of the game's current state. It also decides which power-up to buy every 5 seconds to scale efficiency by selecting the most expensive one available.
